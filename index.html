Packages

Rarimo JavaScript SDK is a library that consists of many smaller npm packages within the @rarimo namespace, a so-called monorepo. Here are some of them to get an idea:

Package Description Latest
@rarimo/provider A package which wraps managing multiple (EVM and non EVM type) chains wallet extension providers to the one common interface.
@rarimo/nft-checkout A package with cross-chain NFT purchases functionality, based on Rarimo Protocol.

Webpack Configs

With Webpack 4 you don't need to do anything, but for Webpack 5 you have to add the following to your Webpack config:

module.exports = [
  {
    // ...
    plugins: [
      ...,
      new webpack.ProvidePlugin({
        Buffer: ["buffer", "Buffer"],
      }),
    ],
    // ...
    resolve: {
      fallback: {
        buffer: false,
        crypto: false,
        events: false,
        path: false,
        stream: false,
        string_decoder: false,
      },
    },
  },
]
